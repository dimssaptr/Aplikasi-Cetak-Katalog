# Aplikasi Cetak Katalog Induk & Label Perpustakaan

Aplikasi berbasis web sederhana (HTML/JS/CSS) untuk membantu pustakawan dalam proses inventarisasi bahan pustaka. Aplikasi ini dirancang untuk menghasilkan output cetak yang presisi untuk:
1. **Kartu Induk** (Ukuran standar 9.5 x 12 cm).
2. **Label Punggung** (Otomatis mendeteksi warna klasifikasi DDC).
3. **Barcode** (Code 128).

## Fitur Utama
* **Client-Side Only:** Tidak perlu instalasi server (XAMPP/MySQL). Cukup buka di browser.
* **Auto-Layout:** Format isian otomatis menyesuaikan template Kartu Induk.
* **Smart Label:** Warna label otomatis berubah sesuai kelas utama DDC (Misal: 600 = Merah, 000 = Coklat).
* **Print Modes:** Tersedia opsi cetak terpisah (Kartu saja, Label saja, Barcode saja) atau Gabungan (Label & Barcode sejajar).

## Cara Penggunaan
1.  Kunjungi link: https://dimssaptr.github.io/Aplikasi-Cetak-Katalog/
2.  Isi metadata buku pada form yang tersedia.
3.  Klik tombol cetak sesuai kebutuhan.
4.  Gunakan kertas ukuran A4 atau F4 pada printer, lalu potong sesuai garis panduan.

## Tentang Pengembang
Aplikasi ini dikembangkan sebagai alat bantu praktikum dan operasional perpustakaan.
